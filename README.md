# NSDC-Spotify-Popularity-Prediction-Project
Predicting the popularity of Tyler the Creator's new album: _Chromakopia_

As part of a group of 8 people we created this project as part of a data science club at UCLA called the National Student Data Corps, or NSDC. Our project seeks to analyze past Spotify data to predict the popularity of Tyler the Creator's new album called _Chromakopia_.
